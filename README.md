🚀Career Development Web App

A complete Career Development Platform designed to help users improve their skills, prepare for interviews, and track their progress — all in one place.

🌐 Live Features
Live Demo : https://ksaikumar9513.github.io/Job-Bridge/  </br>
This project includes multiple modules to guide users through their career journey:

📄 Pages Included
index.html → Home page with project overview
login.html → User authentication page
dashboard.html → User dashboard with progress tracking
mockinterview.html → Practice interview questions
mocktest.html → Take skill-based tests
resumepreview.html → Resume preview & design
skillroadmap.html → Step-by-step learning path
carrerpath.html → Career guidance and role-based paths
📌 Project Description

This web application helps users:

🎯 Identify career paths
🧠 Practice mock interviews
📝 Take skill assessments
📄 Build and preview resumes
📊 Track learning progress

It provides a structured roadmap for students and job seekers to improve their technical and professional skills.

🛠️ Tech Stack
HTML5 – Structure
CSS3 – Styling & Layout
JavaScript (ES6) – Interactivity
